  • Double Langue

    Projet Java de premier semestre. Nous devions créer un jeu ludo-pédagogique en tour par tour via un terminal. Notre travail, effectué en binôme, s'est tourné vers un concept de traduction français-anglais avec un mode histoire et un mode entraînement. Le code est open-source, vous pouvez le consulter sans soucis !
